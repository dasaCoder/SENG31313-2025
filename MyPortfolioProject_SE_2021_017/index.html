My Latest Projects

LuxeAura Cosmetic website

Luxeaura Cosmetics is an e-commerce website developed as a group project to sell a wide range of cosmetic products. The website features a user-friendly interface designed using HTML, CSS, and JavaScript for the frontend, with PHP handling backend functionality and MySQL for database management. This project demonstrates our ability to build a fully functional online store with product listings, user management, and order handling.

FuelQuota management system

Fuel Quota Management System is a group project developed as a solution to efficiently manage fuel distribution during the COVID-19 crisis. The system allows fuel stations to scan QR codes and allocate fuel based on consumers' remaining weekly quota. It was built using React.js for the frontend, Spring Boot for the backend, MySQL for the database, and Spring Security for authentication and access control.
